## Java Scriptとは

**1. 基本的なこと**
- Webアプリ開発にて必須の存在
- 現在、ほぼ全てのサイトで使用されている

**2. プログラムの実行**
```script
console.log("Hello World");
```
【出力結果】
>_ コンソール
Hello world

メモ：
- コンソールという画面は出力結果を出してくれる
- "console.log(~);"というコードを書くと、()の~という文字がコンソールに出力される
- Hello worldのような文字を文字列と呼ぶ
- 文字列はシングルクォーテーションもしくはダブルクォーテーションで囲む必要がある
- 文の最後はセミコロン（;）で終わる。なくてもいいが、ある方がエラーが出にくくなる
  
**3. 数値や計算**
```script.js
console.log(2);
console.log(8+4);
console.log(7-4);
```
